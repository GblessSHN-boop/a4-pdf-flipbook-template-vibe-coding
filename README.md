# E-Brochure PDF Flipbook Template Vibe Coding

## English

**E-Brochure PDF Flipbook Template** is a static frontend template for displaying A4 PDF documents as an interactive digital flipbook. This project is designed for digital brochures, e-books, catalogs, product booklets, company profiles, campus information pages, registration brochures, and other document based publication needs.

This template provides a clean homepage, A4 PDF preview, dedicated brochure viewer page, download access, static brochure data, and a reusable structure for future backend or admin upload integration.

The current version focuses on a simple, stable, and easy to modify frontend foundation. Users can replace the sample PDF files, edit brochure metadata, adjust the visual style, and use the project as a starting point for a digital publication interface.

## Indonesia

**E-Brochure PDF Flipbook Template** adalah template frontend statis untuk menampilkan dokumen PDF ukuran A4 dalam bentuk flipbook digital interaktif. Project ini cocok untuk brosur digital, e-book, katalog, booklet produk, profil perusahaan, halaman informasi kampus, brosur pendaftaran, dan kebutuhan publikasi berbasis dokumen.

Template ini menyediakan homepage yang bersih, preview PDF A4, halaman viewer khusus, akses download, data brosur statis, dan struktur yang dapat digunakan ulang untuk pengembangan backend atau admin upload di masa depan.

Versi saat ini berfokus pada fondasi frontend yang sederhana, stabil, dan mudah dimodifikasi. Pengguna dapat mengganti file PDF contoh, mengubah metadata brosur, menyesuaikan tampilan, dan memakai project ini sebagai dasar antarmuka publikasi digital.

## What is This Template?

E-Brochure PDF Flipbook Template helps developers display PDF brochures in a more interactive way. Instead of only opening a plain PDF file, users can browse a homepage, view a brochure card, click the brochure preview, and read the document through a dedicated viewer page.

This template answers three main needs.

```text
How can an A4 PDF brochure be displayed like a digital book?
How can a brochure preview be shown on a clean frontend page?
How can this template become a base for future admin upload integration?
```

The goal is to provide a practical frontend foundation for digital publication experiments without requiring a backend in the first version.

## How to Replace the Brochure

### English

Place your PDF files inside:

```text
assets/brosur
```

Example files:

```text
company-profile-hal-1.pdf
company-profile-hal-2.pdf
company-profile-hal-3.pdf
company-profile.pdf
```

Then update:

```text
data/brochures.js
```

Example configuration:

```javascript
window.BROCHURES = [
  {
    id: "company-profile",
    title: "Company Profile",
    subtitle: "Digital Company Profile",
    description: "A4 PDF company profile with digital flipbook preview.",
    category: "Company Profile",
    author: "Your Organization",
    keywords: ["company profile", "digital brochure", "PDF A4", "flipbook"],
    cover: "assets/brosur/company-profile-hal-1.pdf",
    download: "assets/brosur/company-profile.pdf",
    pages: [
      "assets/brosur/company-profile-hal-1.pdf",
      "assets/brosur/company-profile-hal-2.pdf",
      "assets/brosur/company-profile-hal-3.pdf",
      "assets/brosur/company-profile-hal-3.pdf"
    ]
  }
];
```

Important note:

```text
If the number of pages is odd, duplicate the last page in the pages array.
This keeps the flipbook navigation more stable.
```

### Indonesia

Masukkan file PDF ke folder:

```text
assets/brosur
```

Contoh file:

## PDF Preparation Guide

### English

Recommended PDF settings:

```text
Document size : A4
Orientation   : Portrait
Export mode   : High Quality Print
Image quality : High
Recommended   : 300 DPI for image based brochures
Security      : No password, no encryption
```

Avoid low quality screenshots. If the source PDF is already blurry, the browser cannot fully restore the quality.

### Indonesia

Pengaturan PDF yang disarankan:

```text
Ukuran dokumen : A4
Orientasi      : Portrait
Mode export    : High Quality Print
Kualitas gambar: High
Rekomendasi    : 300 DPI untuk brosur berbasis gambar
Keamanan       : Tanpa password, tanpa enkripsi
```

Hindari screenshot berkualitas rendah. Jika PDF sumber sudah pecah, browser tidak bisa sepenuhnya memperbaiki kualitasnya.

## Troubleshooting

### Cover preview does not appear

Check whether the PDF path in `data/brochures.js` matches the file name inside `assets/brosur`.

```text
cover: "assets/brosur/uji-coba-brosur-hal-1.pdf"
```

### Browser still shows old version

Use hard refresh.

```text
Ctrl + Shift + R
```

### PDF opens directly but not in viewer

Check `js/app.js`, `js/viewer.js`, and browser Console.

```text
F12
Console
```

### PDF looks blurry

Export the source document again with high quality print settings. Use A4 size and high image quality.
